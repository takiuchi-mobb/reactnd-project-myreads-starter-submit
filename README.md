# Getting started
You can run this code as following commands.
- npm install
- npm run start
- Open browser to http://localhost:3000

# Dependencies
This code is using as below libraries:
- react
- react-dom
- react-router-dom
- escape-string-regexp
- sort-by
- lodash
- prop-types
